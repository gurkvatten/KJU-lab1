# F1 Database

School assignment built with Spring Boot and Thymeleaf.

## Requirements

- Docker Desktop
- Java 25

## How to run

1. Start the database:
```bash
   docker compose up -d
```

2. Run the application in IntelliJ

3. Go to `http://localhost:8080` in your browser

4. Or just check out `https://kju-lab1-production.up.railway.app`

## That's it

Seed data is loaded automatically so the app comes with drivers and tracks out of the box.
